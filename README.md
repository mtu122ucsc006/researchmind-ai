# ResearchMind AI

AI Research Development Platform with a frontend React app and a Node.js backend.

## Install and run

### 1. Install frontend dependencies
```bash
cd "C:\Users\harini\awt\myexpressapp\researchmind-ai"
npm install
```

### 2. Install backend dependencies
```bash
cd server
npm install
```

### 3. Start the backend server
```bash
cd server
npm run dev
```

### 4. Start the frontend
```bash
cd ..
npm run dev
```

### 5. Open in browser
```text
http://localhost:5173
```
